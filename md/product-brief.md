# Math Helper 产品简报

> 高等数学可视化学习工具 - 让抽象数学变得直观易懂

## 1. 产品愿景

**一句话描述**: 一款面向大学生和考研群体的高等数学学习桌面应用，通过交互式可视化和AI辅助，让抽象的数学概念变得直观易懂。

**核心价值主张**:
- 🎯 **看得见的数学** - 将抽象公式转化为动态可视化
- 💡 **听得懂的概念** - AI生成生活化比喻辅助理解
- 📚 **理得清的体系** - 完整知识图谱构建系统认知

## 2. 目标用户

### 主要用户群
| 用户群 | 特征 | 核心需求 |
|--------|------|----------|
| **大学生** | 正在学习高等数学课程 | 理解抽象概念，通过期末考试 |
| **考研学生** | 需要系统复习高数 | 深入理解，掌握解题技巧 |

### 用户痛点
1. **抽象难懂** - ε-δ语言等概念缺乏直觉
2. **公式易混** - 公式多且相似，难以记忆
3. **缺乏可视化** - 课本静态，无法想象动态过程
4. **知识割裂** - 各章节联系不清，缺乏体系

## 3. 功能模块

### 模块A: 函数绘图引擎
| ID | 功能 | 描述 | 优先级 |

## 5. 技术架构

### 技术栈
| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 桌面框架 | Electron | 跨平台桌面应用框架 |
| 前端框架 | Vue 3 + TypeScript | 组合式API，类型安全 |
| 状态管理 | Pinia | Vue 3官方推荐 |
| UI组件库 | Element Plus | Vue 3生态首选 |
| 图表库 | ECharts | 常规图表与基础可视化 |
| 动画库 | D3.js | 复杂交互动画 |
| 公式渲染 | KaTeX | 高性能数学公式渲染 |
| 公式解析 | Math.js | 数学表达式解析与计算 |
| AI服务 | 通义千问 API | 比喻生成与智能问答 |

## 9. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| AI比喻质量不稳定 | 用户体验差 | 预设精选比喻为主，AI补充 |
| 复杂动画性能问题 | 卡顿 | 使用Canvas/WebGL渲染 |
| Electron包体积大 | 下载慢 | 后续考虑迁移Tauri |
